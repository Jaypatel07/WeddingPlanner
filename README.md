# WeddingPlanner
Web Application to plan a wedding and RSVP to other weddings

# Features
Login / Registration <br>
Add Wedding <br>
RSVP to other users weddings <br>
Un-RSVP to other users weddings <br>
Update Wedding <br>
Remove Wedding <br>

# Built Using
HTML5 <br>
CSS3 <br>
C# <br>
.NET Core <br>
Entity Framework<br>
MySQL <br>
